# StellarTrail MVP

## Goal

Build a WeChat Mini Program-first outdoor assistant for China routes, gear planning, and field skills.

## First release scope

1. WeChat Mini Program login and account model.
2. Mountain and route catalog.
3. Route detail with difficulty, season, risk, transport, and gear suggestions.
4. User gear library.
5. Route-based packing checklist generation.
6. Skill catalog focused on knots, camping, packing, navigation, weather, and first aid.
7. Content importer from YAML/Markdown.

## Explicitly out of scope for MVP

- Realtime navigation.
- Social feed/community.
- Route marketplace or guided trips.
- Full GPX editing.
- Storefront/commerce.
